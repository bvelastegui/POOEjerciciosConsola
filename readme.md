# EJERCICIOS DE JAVA POR CONSOLA

### INDICACIONES GENERALES:

1. Crear un proyecto Java en IntelliJ IDEA.
2. Crear una clase por ejercicio.
3. Usar el método main.
4. Utilizar Scanner para el ingreso de datos.
5. Mostrar resultados con System.out.println.
6. Ejecutar el programa desde la consola.

## ✅ EJERCICIO 1 – Registro básico de persona

Desarrollar un programa que permita ingresar nombre, edad y ciudad. Mostrar los datos
ingresados de forma ordenada.

## EJERCICIO 4 – Cálculo de salario . Utilizar métodos

Ingresar horas trabajadas y valor por hora. Calcular y mostrar el salario total.

## EJERCICIO 5 – Área de figuras

Ingresar base y altura. Calcular el área del rectángulo y del triángulo.

Ingresar temperatura en Celsius y convertirla a Fahrenheit.

## EJERCICIO 9 – Total de compra- Utilizar métodos
